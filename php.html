📌 ¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación de propósito general, diseñado especialmente para el desarrollo web. Se ejecuta del lado del servidor y permite generar contenido dinámico, gestionar sesiones, interactuar con bases de datos y más.

🔤 Sintaxis y estructura

El código PHP se inserta entre etiquetas <?php ... ?>. Es sensible a mayúsculas, permite declarar variables con $, utilizar condicionales (if, else) y bucles (for, while).

🗂️ Variables y tipos de datos

PHP es débilmente tipado. Soporta tipos como enteros, cadenas, booleanos, arreglos y objetos. Las variables se declaran con $nombre.

📬 Formularios y entrada del usuario

PHP se usa comúnmente para procesar formularios HTML, accediendo a los datos mediante $_POST o $_GET. Es importante validar y sanitizar esta entrada para evitar inyecciones o errores.

🔐 Cookies y sesiones

Las cookies almacenan información en el navegador del usuario. Las sesiones permiten conservar datos entre páginas del mismo sitio. Se gestionan con session_start() y la variable global $_SESSION.

🧩 Funciones y reutilización

Las funciones en PHP se declaran con la palabra clave function. También se pueden usar funciones predefinidas y bibliotecas externas para ampliar funcionalidades.

📦 PHP orientado a objetos

PHP permite usar clases, herencia, métodos, atributos y constructores. Esto facilita la modularidad, reutilización y escalabilidad de los sistemas.

⚠️ Manejo de errores

Puedes usar estructuras try-catch para capturar errores y excepciones. Es útil para evitar que una falla detenga todo el sistema.

🔧 Uso con bases de datos

PHP se conecta fácilmente con MySQL mediante extensiones como mysqli o PDO. Permite insertar, consultar, actualizar y eliminar datos desde scripts del lado del servidor.
