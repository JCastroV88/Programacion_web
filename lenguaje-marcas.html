📌 ¿Qué es un lenguaje de marcas?

Es un lenguaje diseñado para organizar y estructurar información. Usa etiquetas o marcas que indican la función de cada fragmento de texto. No se ejecuta como un programa, sino que indica cómo debe interpretarse el contenido.

📄 Tipos de lenguajes de marcas

  • De presentación: Marcan directamente cómo debe visualizarse el contenido (ej. RTF).
  • Descriptivos: Describen la estructura o contenido sin indicar cómo se debe ver (ej. HTML, XML).

🔖 Características generales

  • Uso extensivo de etiquetas.
  • Son legibles por humanos y máquinas.
  • Separan contenido de formato (en la mayoría de los casos).
  • Son ampliamente usados en documentos web, científicos, técnicos, etc.

🌐 HTML y XML: los más conocidos

HTML (HyperText Markup Language) es el lenguaje base de la web, define la estructura del contenido que vemos en el navegador. XML (eXtensible Markup Language) permite definir lenguajes personalizados para almacenar y transportar datos.

🧱 Estructura de un documento HTML

Un documento HTML se compone de elementos organizados en etiquetas: <html>, <head>, <title>, <body>, entre otros. Estas etiquetas definen la jerarquía y presentación del contenido.

⚙️ Beneficios del uso de lenguajes de marcas

  • Facilitan la interoperabilidad entre sistemas.
  • Permiten separar contenido, estilo y comportamiento.
  • Son esenciales para la accesibilidad y la indexación por motores de búsqueda.
  • Fomentan el uso de estándares abiertos.
